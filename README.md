### 2048 in Go language
---
Rules [here](https://en.wikipedia.org/wiki/2048_(video_game)).

##### Keys:  
Play with 'w', 'a', 's', 'd'.  
'w' to slide up.  
'a' to slide left.  
's' to slide down.  
'd' to slide right.  
'q' to quit the game.  

##### Requirements:  
Download the go package [goterm](https://github.com/buger/goterm).  
`go get -u github.com/buger/goterm`

##### Usage:
```
git clone https://github.com/aditya1904/2048go  
cd 2048go  
go build -o 2048  
./2048
```
